# finger-camera
A finger detection using camera

## How to use:
This code uses [OpenCV](https://opencv.org/) for camera and [mediapipe](https://ai.google.dev/edge/mediapipe/solutions/guide) for hand gesture detection.

If needed for app use just open the `main.py` file.
But if you need the hand gesture in a code you can use `client.py` file.

If you need to add gestures use `gestures.py` file.
